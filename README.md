# Crypto API

A RESTful API built with Python to fetch cryptocurrency data, including coin listings, categories, and market data in INR and CAD. The project is Dockerized for easy deployment.

---

## **Features**

- List all coins with their IDs.
- List coin categories.
- Fetch specific coins by ID or category.
- Market data available in **INR (Indian Rupee)** and **CAD (Canadian Dollar)**.
- Paginated API responses using `page_num` query parameter.
- Dockerized for easy setup and deployment.

---

## **Tech Stack**

- **Backend:** Python (Flask/FastAPI)
- **Data:** Coin data from APIs
- **Containerization:** Docker, Docker Compose
- **Version Control:** Git & GitHub
